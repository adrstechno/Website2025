## Image Sources

You can get professional images from:
- **Unsplash** (https://unsplash.com) - Free high-quality photos
- **Pexels** (https://pexels.com) - Free stock photos
- **Your own photos** - Professional headshots and office photos

## Search Keywords for Stock Photos

- Hero: "cloud computing", "data center", "enterprise technology", "modern office"
- Services: "dashboard ui", "cloud server", "mobile app development", "ai technology"
- Products: "software interface", "web dashboard", "app mockup"
- About: "office space", "team meeting", "modern workspace"
- Team: Professional headshots or use placeholder initials

## Placeholder Option

If you don't have images yet, the website will show:
- Colored backgrounds with initials for team members
- Icon placeholders for features
- The site will still look professional

## After Adding Images

1. Place all images in the correct folders
2. Ensure file names match exactly (case-sensitive)
3. Optimize images for web (compress if needed)
4. Recommended formats: JPG for photos, PNG for logos/graphics
5. Keep file sizes under 500KB for faster loading

---

**Note:** All image paths are already configured in the code. Just add the images to the public folder and they will automatically appear!
